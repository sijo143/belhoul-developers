# Belhoul Developers - Elegant Coming Soon Page

An ultra-minimalist, high-end "Coming Soon" landing page featuring responsive structures, a subtle cinematic pan-zoom skyline backdrop, and an interactive gold dust particle canvas.

## Directory Structure

```text
belhoul-coming-soon/
├── index.html                  # Main HTML presentation document
├── README.md                   # Setup documentation
├── Logo.png                    # Brand Logo (Primary asset)
├── bg.png                      # Skyline Background Image (Subtle 0.18 opacity)
└── assets/
    ├── css/
    │   └── style.css           # Premium typography and custom animations
    └── js/
        └── main.js             # Canvas particle engine & page interactions
```

## Setup Instructions

1. Place your background skyline file (`bg.png`) and brand logo (`Logo.png`) into the root directory next to `index.html`.
2. Open `index.html` in any web browser to test locally.
3. This project is fully responsive and compatible with GitHub Pages, Netlify, or custom servers.
